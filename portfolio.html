Project Portfolio

PacMen Exercise: is an exercise that contains code for building a feature of the PacMan game, displaying and moving a PacMan from one side of the screen to the other.

Eye Exercise: is an exercise that has mouse movement code to update the position of the "eye" or "eye" elements displayed on DOM or in other words the Browser Window.

Real Time Bus Tracking Exercise: is an exercise that contains refactored code for Real Time Bus Tracker, tracking the path of the bus between MIT and Harvard Universities. Refactored code includes API code from MapBox in Dark Mode which is exciting!

ADDITIONAL PROJECT INCLUDED: Factory PacMen Exercise: is an exercise for you to create multiple PacMan or PacMen with the click of a button by using array methods. Each of the PacMan will move about the browser window at their own pace.
